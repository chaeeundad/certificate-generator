=====================================
교육 이수증 자동 생성 프로그램 사용 가이드
=====================================

[프로그램 설치 없이 바로 사용하기]

Windows 사용자:
1. "교육이수증생성기.exe" 파일을 더블클릭하여 실행합니다.
2. 바이러스 백신 경고가 나타날 수 있습니다.
   "추가 정보" → "실행"을 클릭하세요.

Mac 사용자:
1. "교육이수증생성기" 파일을 더블클릭하여 실행합니다.
2. 보안 경고가 나타나면 "시스템 환경설정" → "보안 및 개인 정보 보호"에서
   앱 실행을 허용해주세요.

=====================================
[프로그램 사용 방법]

1단계: 템플릿 파일 준비
- 제공된 "교육이수증_템플릿.hwpx" 파일을 사용하거나
- 직접 만든 템플릿을 사용할 수 있습니다.

템플릿에 포함해야 할 텍스트:
- {이름} : 교육생 이름이 들어갈 자리
- {생년월일} : 생년월일이 들어갈 자리
- {부서} : 소속 부서가 들어갈 자리
- {교육과정명} : 교육 과정 이름이 들어갈 자리
- {교육기간} : 교육 기간이 들어갈 자리
- {교육시간} : 총 교육 시간이 들어갈 자리
- {발급일자} : 이수증 발급 날짜가 들어갈 자리
- {발급기관} : 발급 기관명이 들어갈 자리
- {번호} : 이수증 번호가 들어갈 자리

2단계: CSV 파일 준비
- 엑셀에서 다음 형식으로 작성 후 CSV로 저장:

  이름,생년월일,부서
  홍길동,1990.01.15,개발팀
  김철수,1985.05.20,마케팅팀
  이영희,1992.11.30,인사팀

3단계: 프로그램 실행
1. "템플릿 선택" 버튼 클릭 → HWPX 템플릿 파일 선택
2. "CSV 선택" 버튼 클릭 → 교육생 명단 CSV 파일 선택
3. "폴더 선택" 버튼 클릭 → 이수증을 저장할 폴더 선택
4. 교육 정보 입력:
   - 교육과정명: (예: 2024년 정보보안 교육)
   - 교육기간: (예: 2024.09.01 ~ 2024.09.30)
   - 교육시간: (예: 40)
   - 발급일자: (자동으로 오늘 날짜가 입력됨)
   - 발급기관: (예: 주식회사 ABC)
5. "이수증 생성" 버튼 클릭

4단계: 결과 확인
- 선택한 폴더에 "이수증_날짜시간" 폴더가 생성됩니다.
- 각 교육생별로 "이수증_001_홍길동.hwpx" 형식의 파일이 생성됩니다.

=====================================
[문제 해결]

Q: 프로그램이 실행되지 않습니다.
A: Windows Defender나 바이러스 백신에서 차단할 수 있습니다.
   예외 항목으로 추가하거나 일시적으로 비활성화 후 실행하세요.

Q: CSV 파일을 읽을 수 없다고 나옵니다.
A: CSV 파일을 메모장으로 열어서 UTF-8로 다시 저장해보세요.
   파일 → 다른 이름으로 저장 → 인코딩을 "UTF-8"로 변경

Q: 생성된 HWPX 파일이 열리지 않습니다.
A: 한글 프로그램이 설치되어 있어야 합니다.
   한컴오피스 또는 한글 뷰어를 설치하세요.

Q: 템플릿의 텍스트가 바뀌지 않습니다.
A: 템플릿 파일의 {이름} 같은 플레이스홀더가
   정확히 중괄호 포함하여 입력되었는지 확인하세요.

=====================================
[참고사항]

- 대량 생성 시 시간이 걸릴 수 있습니다. (100명 기준 약 1-2분)
- 템플릿 파일은 원본이 보존되므로 안심하고 사용하세요.
- 생성된 이수증은 날짜시간별 폴더로 구분되어 저장됩니다.
- CSV 파일의 이름은 한글, 영문 모두 가능합니다.
